# Misskey-SMTC

Windows 10/11環境で、今聴いている曲をMisskeyに投稿するためのツール

WinUI3なので見た目はいいです　多分

Appleとは一切関係がありません。

## 最低要件

- Windows 10 (ver 1809, 17763)
- .NET 10
- (Misskeyに投稿する場合)Misskey v12.27.0以降のサーバー
  - MiAuthを使用してトークンの取得を行います。
  - 要望があった場合手動でのアクセストークン登録も追加します

## ビルド

- Visual Studio 2026 Community

まだパッケージごと配布する準備ができてないので各自ビルドしてください。

`Misskey-SMTC (Package)`を右クリックして`公開` -> `アプリ パッケージの作成...`と進んであとはいい感じにしてください。
MSIXパッケージが生成されるので同時に生成されるps1を実行すればインストールされます　多分
